About

Memory Match is a fun memory game using cards.

There are 16 cards on the deck, each with a matching pair. The goal is to find all the pairs in the quickest time possible and with the lowest number of flips.

Currently there is only 1 set of cards but I will be adding more every now and then.

The game was developed as part of an assignment for the Udacity Web Developer course.
